Development Approach

This projects is motivated by my fascination with IoT electronics and my desire to monitor the status of my apartment 24/7. It was incredibly fun to put all this together as it required such a diverse set of tools (fullstack, database, microcontrollers and sensors). You are looking at the dashboard for my IoT devices hosted on a RaspberryPi, including a thermister to monitor temperature and an accelerometer to monitor activity (which, as you'd imagine, is basically a very good random number generator).

The front end is built with Bootstrap and jQuery based on my modified version of the SB Admin 2 theme. The backend is built with Node.js and Socket.io, while data is stored in a SQLite database. The server is hosted with NGINX on a Raspberry Pi 3B+ in my bedroom, which also controls the sensors through an Arduino. In the future I will implement the MQTT protocol to communicate with multiple IoT devices.
