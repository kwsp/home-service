Development Approach

This website is a dashboard for my IoT devices at home. Current this includes a thermister to monitor temperature and an accelerometer to monitor activity (which, as you'd imagine, is basically a very good random number generator). The list will surely expand in the future.

I built this website mostly for fun but also as a fullstack project. The front end is built with Bootstrap and jQuery based on my modified version of the SB Admin 2 theme. The backend is built with Node.js and Socket.io is used to pass data between the server and the client. The server is hosted with NGINX on a Raspberry Pi 3B+ in my bedroom, which also controls the sensors through an Arduino. In the future I will implement the MQTT protocol to communicate with multiple IoT devices.
